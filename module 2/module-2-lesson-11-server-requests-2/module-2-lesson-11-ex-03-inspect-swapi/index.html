Make requests to the SWAPI and get information from the Network tab to answer the following questions:

  • where is the request method
      clicking on the Name, then Headers
  • the response code (remember that 200 is OK)
      also Headers, but you can also see it in Status
  • in the request headers look for a user-agent call, what can you tell about its content?
      This user agent string belongs to Chrome browser running on Android OS. The browser is developed by Google Inc and renders web pages using the Blink engine.
